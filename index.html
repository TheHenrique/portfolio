Portfólio

Henrique Lustosa

Exploração visual, responsividade e desenvolvimento front-end e back-end.

01 — Front-end

EBAC Shoes

Desenvolvimento de interface para loja virtual com tema escuro. O projeto apresenta vitrine dinâmica de produtos, seções institucionais e botões de interação para o carrinho de compras.
02 — Portifólio

Portfólio Pessoal Dinâmico

Projeto de portfólio anterior com design minimalista e tema escuro. O grande destaque é a animação de texto na página inicial, que alterna automaticamente as áreas de atuação, criando uma introdução moderna e interativa.
03 — Repositório Técnico

Galeria de Projetos Pessoais

Desenvolvimento de página responsiva focada na organização de múltiplos conteúdos. O layout utiliza componentes estruturais clássicos, como carrossel de destaques e um sistema de grades para apresentar simulações técnicas e análises de forma clara e acessível.
04 — Web App & Lógica

Fuel Estimator

Desenvolvimento de uma ferramenta interativa para o cálculo de consumo de combustível em corridas. A aplicação combina lógica de programação com uma interface funcional, permitindo a inserção rápida de variáveis de tempo e capacidade para entregar estimativas precisas de pitstops e litragem.
05 — Web Design Institucional

Website Gastronômico

Desenvolvimento de interface institucional para o setor de restaurantes. O layout utiliza um carrossel de imagens em destaque e tipografia estilizada para transmitir a atmosfera do local, organizando de forma clara as seções essenciais como sobre o espaço, eventos e cardápio.
06 — Landing Page & Interação

Promoção de Eventos Tech

Criação de uma landing page promocional focada em conversão. O grande diferencial técnico é a implementação de um cronômetro regressivo dinâmico para criar senso de urgência, integrado a um painel de informações claras sobre local, data e preço, direcionando o usuário para a chamada de ação (CTA).
07 — Web App & UI Design

Plataforma Efood

Desenvolvimento de interface estilo marketplace voltada para o setor gastronômico. O layout organiza diferentes estabelecimentos em cartões estruturados, trazendo tags de categorias, notas de avaliação e descrições customizadas para cada restaurante.
08 — Web App & Produtividade

Gerenciador de Tarefas Avançado

Desenvolvimento de uma aplicação interativa voltada para foco e produtividade. O sistema permite o gerenciamento completo de atividades com recursos de categorização, busca e filtragem por prioridade, facilitando a organização de rotinas e o acompanhamento visual das metas concluídas.
09 — Front-end & Interface

Portal de Vagas Tech

Desenvolvimento de uma plataforma de busca de oportunidades para profissionais de tecnologia e design. A interface apresenta um sistema de pesquisa focado e organiza as vagas de forma clara, destacando dados fundamentais para o candidato, como requisitos técnicos, salário e modelo de contratação.
10 — Front-end & ReactJS

Calculadora de IMC

Aplicação web interativa desenvolvida em ReactJS. O projeto apresenta uma interface minimalista focada na usabilidade, combinando a captura de dados do usuário com lógica matemática para processar e retornar o Índice de Massa Corporal de forma rápida e eficiente.
