# Portfólio Pessoal | Henrique Lustosa

Um portfólio web minimalista e moderno, desenvolvido com foco em performance, responsividade e estética *dark mode*. Este projeto serve como vitrine para minhas aplicações front-end, explorando construção de interfaces, precisão técnica e UI/UX.

## 🚀 Visão Geral

O design deste portfólio foi construído utilizando tecnologias nativas (HTML e CSS), priorizando código limpo, estruturação semântica e animações fluídas (hover effects) para proporcionar a melhor experiência ao usuário, sem depender de bibliotecas externas pesadas.

## ✨ Funcionalidades e Destaques

- **Design Responsivo:** Layout 100% adaptável para qualquer tamanho de tela (Desktop, Tablet e Mobile) utilizando conceitos modernos de CSS Grid e Flexbox.
- **Tema Dark (Dark Mode):** Paleta de cores sofisticada focada em contraste, legibilidade e conforto visual, utilizando variáveis CSS (`:root`) para fácil manutenção.
- **Animações e Micro-interações:** Efeitos suaves de elevação (lift) e zoom nas imagens ao interagir com os cartões dos projetos.
- **Performance:** Uso de ícones em SVG inline e imagens otimizadas para garantir um carregamento rápido, evitando requisições desnecessárias.

## 🛠️ Tecnologias Utilizadas

**Na construção do portfólio:**
- **HTML5:** Estrutura semântica.
- **CSS3:** Variáveis nativas, Grid Layout, Flexbox, Media Queries e Transitions.

**Nos projetos demonstrados na galeria:**
- JavaScript 
- ReactJS
- CSS-in-JS
- Ferramentas de Front-end

## 💻 Galeria de Projetos

O portfólio exibe projetos práticos, abrangendo diferentes desafios do desenvolvimento web:
- **Landing Pages:** Focadas em conversão com elementos dinâmicos (como cronômetros).
- **Aplicações Web e SPAs:** Como calculadoras em ReactJS e Listas de Tarefas interativas.
- **Plataformas de Listagem:** Simulando integrações reais (Portais de Vagas e Delivery Gastronômico).
- **Sites Institucionais:** Focados na apresentação visual e identidade de marca.
